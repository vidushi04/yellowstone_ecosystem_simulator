# Card images for ecosystem components

Drop image files into this folder. **Use these exact filenames** (including extension) so the main simulator can show them on each card.

Supported formats: **.png** or **.jpg**

## Filenames that match the code

| Component ID (code) | Card label      | Use this filename |
|--------------------|-----------------|--------------------|
| Wolves             | Wolves          | `Wolves.png` or `Wolves.jpg` |
| Elk                | Elk             | `Elk.png` or `Elk.jpg` |
| CottonWood         | Cotton Wood     | `CottonWood.png` or `CottonWood.jpg` |
| BerryTrees         | Berry Trees     | `BerryTrees.png` or `BerryTrees.jpg` |
| Grass              | Grass           | `Grass.png` or `Grass.jpg` |
| Bears              | Bears           | `Bears.png` or `Bears.jpg` |
| LandFertility      | Land (fertility)| `LandFertility.png` or `LandFertility.jpg` |
| Birds              | Birds           | `Birds.png` or `Birds.jpg` |
| RiverQuality       | River (quality) | `RiverQuality.png` or `RiverQuality.jpg` |
| Beaver             | Beaver          | `Beaver.png` or `Beaver.jpg` |
| Dam                | Dam             | `Dam.png` or `Dam.jpg` |
| Fish               | Fish            | `Fish.png` or `Fish.jpg` |
| OtherAnimals       | Other Animals   | `OtherAnimals.png` or `OtherAnimals.jpg` |
| Cattle             | Cattle          | `Cattle.png` or `Cattle.jpg` |
| RangersRanchers    | Rangers/Ranchers| `RangersRanchers.png` or `RangersRanchers.jpg` |
| Visitors           | Visitors        | `Visitors.png` or `Visitors.jpg` |
| ParkRevenue        | Park Revenue    | `ParkRevenue.png` or `ParkRevenue.jpg` |

If a file is missing, the card shows the first letter of the component name as a placeholder.
